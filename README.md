## About

BrainTech is a multi-section landing page for a tech agency or startup. Built as an early project using classic web technologies — Bootstrap for responsive layout, jQuery for interactivity, AOS for scroll animations and Isotope for filterable portfolio grid.

🔗 **Live:** [vergos1.github.io/project_OLD-Braintech](https://vergos1.github.io/project_OLD-Braintech/)

## Features

- **Hero section** — full-width banner with call-to-action
- **About** — mission, vision and values
- **Services** — service cards with icons and descriptions
- **Portfolio** — filterable project grid powered by Isotope
- **Team** — team member profiles
- **FAQ** — accordion-style frequently asked questions
- **Scroll animations** — AOS library for reveal-on-scroll effects
- **Carousel** — Owl Carousel for slider sections
- **Lightbox** — Venobox for image gallery overlays
- **Contact form** — PHP Email Form integration
- **Responsive** — fully adaptive layout via Bootstrap grid
